# Group-09-Project-Appointment-Service

## Getting started

1. Run `npm install`
2. Create a `.env` file in the working/root directory and define these variables
    * MONGODB_URL
    * GOOGLE_MAPS_API_KEY
    * MQTT_BROKER_URL
    * MQTT_BROKER_PORT
    * MQTT_BROKER_USERNAME (optional)
    * MQTT_BROKER_PASSWORD (optional)
    * MQTT_CERT_PATH (optional)
    * MQTT_CERT_KEY_PATH (optional)
3. Run `npm run dev` or `npm run start`
